117 ChronoFile

insert context statement

We can often remember when we thought of an idea, based on events that were going on at the same time, better than we can remember abstract ways of filing information.

  • This files things according to when they were created. As a result, it uses one of the four dimensions to organize information.
  • This kind of arrangement also allows you to see the progression of an idea over time. We rarely understand a complicated idea all at once. It usually takes time as we piece together the information that we need to make it relevant to our experience.

Therefore:

File ideas in a chronological order if that seems to work better for you.
